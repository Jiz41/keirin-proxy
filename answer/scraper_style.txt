## scraper.jsにおける選手の脚質取得ロジック

`scraper.js`では、選手の脚質は以下の方法で取得されています。

### 1. HTML要素
レース詳細ページの出走表テーブル(`table`)内の、各選手に対応する行(`tr`)に含まれる`<td>`要素から取得しています。

### 2. セレクタ・コード

まず、`th`に「予想」というテキストを含むテーブルを特定します。

```javascript
  $('table').each((i, table) => {
      const thText = $(table).find('th').first().text();
      if (thText.includes('予想') && !thText.includes('周回')) {
          raceTable = table;
          return false;
      }
  });
```

次に、そのテーブル内の各選手行(`tr`)に含まれる`td`要素のコレクションから、8番目（インデックス`7`）の`<td>`要素のテキストを脚質として取得します。

`indexOffset` という変数で列のインデックスが調整される場合がありますが、脚質の列は `7 + indexOffset` で特定されます。

該当のコードは以下の部分です（159行目）。

```javascript
rider.style = tds[7 + indexOffset] ? $(tds[7 + indexOffset]).text().trim() : '';
```

### 3. 変数名

取得された脚質の文字列は、`rider`オブジェクトの `style` プロパティに格納されます。

```javascript
let rider = {
    // ... 他のプロパティ
    style: '',
    // ... 他のプロパティ
};

// ...

rider.style = $(tds[7 + indexOffset]).text().trim();
```
